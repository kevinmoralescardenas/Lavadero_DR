📖 Descripción
Aplicación de escritorio en Java Swing para gestionar un lavadero de vehículos. Incluye módulos de clientes, servicios, empleados, deudas y autenticación de usuarios, con persistencia en PostgreSQL y arquitectura MVC.

✨ Características
🔐 Login seguro con registro de usuario desde la pantalla inicial.

👥 CRUD completo de clientes (nombre, correo, teléfono).

🛠️ CRUD de servicios (nombre, precio, stock).

👨‍🔧 CRUD de empleados + asignación de servicios.

💰 Módulo de deudas: registrar, listar, pagar y eliminar.

🎨 Interfaz gráfica con menús y tablas interactivas.

🏗️ Tecnologías
Java 17 / Swing

PostgreSQL 12+

JDBC

NetBeans IDE

🚀 Instalación rápida
Clona el repositorio.

Crea la base de datos lavadero_db y ejecuta el script SQL.

Agrega el driver PostgreSQL al proyecto.

Configura credenciales en Conexion.java.

Ejecuta Main.java.

🎮 Uso básico
Inicia sesión con un usuario registrado o créalo desde el botón "Registrar Usuario".

Navega por los menús: Servicios, Usuarios, Empleados, Clientes, Deudas.

Gestiona deudas: listar, pagar, eliminar y registrar.
